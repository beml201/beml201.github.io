Ancestry Plot

A principal component analysis plot of the 1000Genomes project. This plot was done using the flashPCA tool to separate people out by genetic ancestry. These principal components are then used to adjust models, eg GWAS, to account for population structure.

Mendelian Randomisation (MR) Visualisation Tool

MR can be used to assess causative effect, using an instrumental variable and exposure and outcome variables. Genetics can provide a useful instrumental variable, using differing phenotypes eg TSH concentration and birthweight, to assess the causal effect. This tool aims to plot a number of different regression methods as well as a forest plot that can allow the user to interact with it. By enabling zooming in, and removal of certain regressions, it can allow the user to help deal with the data better. The tool can be found here: https://github.com/beml201/MR-Tool

Elbow Method

This tool provides a method that can assign an 'elbow score' to error points generated from a clustering analysis for a series of k clusters. Taking the minimum non-zero elbow score should consistently find the appropriate 'elbow' or near enough. This should be particularly useful for any unclear 'elbows' but should not be used without other means to assess the optimal k. The assumption is that the curve is monotonic, but this should be ok with use of certain error points such as variance explained. Code to be released soon

Protein Structure Based Gene Burden Testing

Rare-variant burden testing can help identify casual genes for a particular phenotype. During my MSc project, I propose that usinf the 3D protein structure which the exomes code for can provide an alternative way to VEP annotated burden testing. As a demonstration, I have taken IGF1R from the AbbVie/Biogen/Broad/Pfizer publicly available GeneBass PheWas. This used using the 450k whole exome sequence release of the UKBiobank data. I have used the phenotype of birth weight for this. The plot links the variants and their genetic position and protein position within 3D space.
